Project Title

Making a houseplant talk

Project Objectives

I will try to make something with an Arduino that will sense the moisture in the plant’s soil, that is, ask the plant if it needs water. If the moisture in the soil is too low, an LCD will display “Water me.”

Features:

  • A device that indicates moisture of a substance.
  • An LCD to display the message clearly.

Implementation Tools:

  1. Arduino
  2. Arduino IDE
  3. Solderless breadboard
  4. An LCD Display
  5. Magic of the Open Source Philosophy
